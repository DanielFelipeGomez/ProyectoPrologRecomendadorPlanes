# ProyectoPrologRecomendadorPlanes
Proyecto desarrollado en Prolog 

Muchas veces deseamos organizar algún plan con nuestra familia, amigos o pareja, pero estamos aburridos de siempre ir a los mismos sitios y 
hacer las mismas actividades. En la asignatura de Inteligencia Artificial se nos ha pedido desarrollar un programa en Prolog con temática libre
en el que podamos demostrar diferentes herramientas que nos ofrece el lenguaje. 

Es así como veo la ocasión perfecta para desarrollar un recomendador de planes que se adecue a determinados parámetros que nos indique 
el usuario. Para el desarrollo de esta práctica necesitaremos usar distintas herramientas de Prolog, como son la entrada y salida complejas, 
el uso de listas y estructuras complejas para almacenar la información.
